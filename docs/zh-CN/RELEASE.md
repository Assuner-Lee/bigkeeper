全局参数如下：

- -u, --user：用户名，默认是 git global config 的 user.name，会显示在命令提示信息中，比如上述提示信息中的默认用户名是 mmoaay

- -p, --path：项目路径名，默认是当前所在的路径

-  -v, --version：项目版本号，默认是当前所在的路径

- home: 发布主工程命令
